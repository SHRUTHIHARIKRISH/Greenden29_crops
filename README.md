# 🌿 Greenden – Responsive Plant Website

## Overview

Greenden is a responsive multi-page website built using **HTML5** and **Tailwind CSS**. The website features a modern, clean, and user-friendly interface for showcasing a collection of indoor and outdoor plants. It demonstrates the use of Tailwind CSS utility classes to create responsive layouts without writing custom CSS.

## Website Structure

The website consists of three main pages:

- **Home Page** – Features a welcoming hero section, featured plants, and an attractive landing page design.
- **Products Page** – Displays a responsive grid of plant products with images, descriptions, and pricing.
- **Contact Page** – Includes a contact form, business information, and location details for user interaction.

## Features

- Responsive navigation bar
- Modern hero section
- Responsive product cards
- Contact form
- Mobile-friendly layout
- Clean and professional UI
- Built using HTML5 and Tailwind CSS

## Responsive Design

The website is fully responsive and adapts to different screen sizes using Tailwind CSS responsive classes (`sm:`, `md:`, `lg:`, and `xl:`). The layout maintains proper spacing, alignment, and typography across mobile, tablet, and desktop devices.

## Learning Objectives

- Build a responsive multi-page website using HTML5 and Tailwind CSS.
- Learn Tailwind CSS utility-first styling.
- Create responsive layouts using Flexbox and Grid.
- Develop clean, maintainable, and modern user interfaces.

## Technologies Used

- HTML5
- Tailwind CSS (CDN)

By completing the Greenden project, a responsive and visually appealing plant-themed website was developed using HTML and Tailwind CSS.
